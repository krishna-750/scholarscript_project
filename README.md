# ScholarScript - Thesis & Dissertation Experts (React Frontend)

## Quick Start
1. Clone the repo
2. Install dependencies: `npm install`
3. Start dev server: `npm start`
4. Build for production: `npm run build`
5. Deploy to Render (see guide)
